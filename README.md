# Truong Huy Research — Weekly Market View

A Bloomberg-dark Astro site that auto-publishes a **Weekly Market View** on the Vietnam stock market (VN-Index), Vietnam macro, and a global cross-asset snapshot. Generated by Python, scheduled by GitHub Actions, deployed on Vercel.

Live: **https://truonghuyresearch.xyz**

## 📊 Weekly Market View Pipeline

Every Saturday at **10:00 UTC** (= 17:00 ICT Friday, after VN market close), GitHub Actions runs `weekly_bot.py`:

1. **Fetch VN-Index** weekly OHLCV, sector performance, foreign net flow via `vnstock`.
2. **Fetch global cross-asset** weekly closes + change %: DXY, Gold, WTI (via `yfinance`); BTC, ETH (via CoinGecko public API).
3. **Generate charts** (matplotlib, Bloomberg-dark theme) → `public/charts/`.
4. **Call LLM** (MiniMax M3 via Ollama Cloud) with the assembled data → structured markdown with YAML frontmatter.
5. **Validate frontmatter** (all required fields, valid `session_tone` enum).
6. **Commit & push** the new `src/content/market-views/YYYY-MM-DD.md`.

Each post contains: Executive Summary, Vietnam Macro Pulse, VN-Index Weekly Review (candlestick chart), Sector Spotlight (rotating), Global Cross-Asset Snapshot, and The Week Ahead. Tone is Bloomberg/FT professional, not retail blog.

### Required environment variables

| Variable            | Default                     | Notes                                  |
| :------------------ | :-------------------------- | :------------------------------------- |
| `MINIMAX_API_KEY`   | —                           | Required. Ollama Cloud key.            |
| `LLM_API_KEY`       | (falls back to `MINIMAX_API_KEY`) | Alias accepted.                |
| `LLM_BASE_URL`      | `https://ollama.com/api`    | Optional.                              |
| `LLM_MODEL`         | `minimax-m3`                | Optional.                              |

For local runs, set `MINIMAX_API_KEY` in your shell or a `.env` file.

## 🛠️ Tech Stack

- **Astro 5.18** — content-driven static site, zero-JS by default.
- **Tailwind CSS** — design tokens for the Bloomberg-dark terminal palette.
- **TypeScript strict** — Zod-validated frontmatter for `market-views` and `research` collections.
- **Python 3.12** — data pipeline + LLM call.
- **vnstock 4.x** — Vietnam market data (VN-Index, sectors, foreign flow).
- **yfinance** — global macro/cross-asset weekly OHLC.
- **CoinGecko public API** — BTC/ETH weekly closes (no key).
- **Matplotlib** — chart generation (Agg backend, dark theme).
- **MiniMax M3 (via Ollama Cloud)** — LLM that drafts the commentary.
