## рџ“Љ Weekly Market View Pipeline

Every Saturday at **10:00 UTC** (= 17:00 ICT Friday, after VN market close), GitHub Actions runs `weekly_bot.py`:

1. **Fetch VN-Index** weekly OHLCV, sector performance, foreign net flow via `vnstock`.
2. **Fetch global cross-asset** weekly closes + change %: DXY, Gold, WTI (via `yfinance`); BTC, ETH (via CoinGecko public API).
3. **Generate charts** (matplotlib, Bloomberg-dark theme) в†’ `public/charts/`.
4. **Call LLM** (MiniMax M3 via Ollama Cloud) with the assembled data в†’ structured markdown with YAML frontmatter.
5. **Validate frontmatter** (all required fields, valid `session_tone` enum).
6. **Commit & push** the new `src/content/market-views/YYYY-MM-DD.md`.

Each post contains: Executive Summary, Vietnam Macro Pulse, VN-Index Weekly Review (candlestick chart), Sector Spotlight (rotating), Global Cross-Asset Snapshot, and The Week Ahead. Tone is Bloomberg/FT professional, not retail blog.

For local runs, set `OLLAMA_API_KEY` in your shell or a `.env` file.
